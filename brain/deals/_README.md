# deals/

One page per engagement (`public.engagements`). Captures stage, value, owner, open threads, and the full timeline of interactions / emails / meetings / stage changes.

## Does NOT go here

- Projects (the things we *do* for a deal) → `projects/`.
- Meetings (individual scheduled events) → `meetings/`.
- Tasks → folded into the deal page's `## Open Threads`.

## Slug rule

`<company-slug>-<short-deal-name>`. E.g., `acme-pilot`, `betatestco-mvp`.
